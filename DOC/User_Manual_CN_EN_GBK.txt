### 3.1 Start-up Process / 开机流程
1. Initialization (1s) / 初始化阶段(1秒):
   - LED1: Flashing (100ms) / LED1闪烁(100ms)
   - LED2: Off / LED2熄灭
   - OLED: Shows init info / 显示初始化信息

2. Warm-up (3s) / 预热阶段(3秒):
   - LED1: Steady on / LED1常亮
   - LED2: Flashing (100ms) / LED2闪烁
   - OLED: Shows countdown / 显示倒计时

3. Working / 工作阶段:
   - LED1: Steady on / LED1常亮
   - LED2: Updates with data / LED2随数据更新
   - OLED: Real-time data / 实时数据显示
